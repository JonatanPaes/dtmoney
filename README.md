# Como Executar?

> Clone o Repositório:

Node version 14.20.0

Clone

```bash
git clone https://github.com/JonatanPaes/dtmoney.git
```

Para executar o projeto

```bash
cd dtmoney
```

Instale as dependências do projeto

```bash
yarn install
```

# Web
Para executar o projeto

```sh
yarn start
```
